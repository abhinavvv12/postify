# 🚀 Postify

Postify is a modern full-stack social media application where users can create, upload, and manage posts with images. Built using the MERN Stack and integrated with ImageKit for image storage and optimization.

## 🌐 Features

- 🔐 User Authentication
- 📝 Create Posts
- 🖼️ Upload Images with ImageKit
- ❤️ Like & Interact with Posts
- 📱 Responsive Design
- ⚡ Fast and Modern UI
- 🔄 Real-Time Updates Ready
- ☁️ Cloud Image Storage

## 🛠️ Tech Stack

### Frontend
- React.js
- CSS3
- Axios

### Backend
- Node.js
- Express.js

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/abhinavvv12/postify.git
cd postify
```

### 2️⃣ Install Dependencies

#### Frontend

```bash
cd client
npm install
```

#### Backend

```bash
cd server
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file inside the server directory.

```env
PORT=5000

MONGODB_URI=your_mongodb_connection_string

IMAGEKIT_PUBLIC_KEY=your_public_key
IMAGEKIT_PRIVATE_KEY=your_private_key
IMAGEKIT_URL_ENDPOINT=your_url_endpoint
```

---

## ▶️ Run Locally

### Start Backend

```bash
cd server
npm run dev
```

### Start Frontend

```bash
cd client
npm run dev
```
